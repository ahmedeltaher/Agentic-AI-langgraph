💰 الأداة الأولى: أسعار اللابتوبات

المدرب بدأ بالأداة الأولى:

🔧 Get Laptop Price Function Tool

🎯 الوظيفة

"This tool is used to find the price of a given laptop"
هذه الأداة تُستخدم لإيجاد سعر لابتوب معين!

💾 قاعدة البيانات

"It uses an RDBMS behind the scenes to keep track of laptop prices"
تستخدم RDBMS في الخلفية لتتبع أسعار اللابتوبات!

🔄 التطبيق العملي

"For our specific illustration, instead of an RDBMS database, we will use an in-memory PANDAS data frame"
للتوضيح المحدد بتاعنا، بدلاً من قاعدة بيانات RDBMS، هنستخدم PANDAS data frame في الذاكرة!

🌍 الاستخدام الحقيقي

"In real world situations, this can access any database, or API, to find the prices"
في المواقف الحقيقية، ده يقدر يوصل لأي قاعدة بيانات، أو API، لإيجاد الأسعار!

📥 المدخل

اسم أو موديل اللابتوب

🔍 العملية

البحث في قاعدة البيانات

📤 المخرج

السعر أو رسالة "غير موجود"

⚡ السرعة

استجابة فورية من الذاكرة

لماذا PANDAS للتوضيح؟
• سهولة الإعداد والاستخدام 🚀
• لا يحتاج إعدادات معقدة 🔧
• سرعة في التجريب 💨
• يمكن تطويره لقواعد بيانات حقيقية لاحقاً 📈

📝 الأداة الثانية: مواصفات المنتجات

المدرب انتقل للأداة الثانية:

📚 Get Product Features Retriever Tool

📖 مصدر البيانات

"This tool is built using a PDF file that contains descriptions of the laptops"
هذه الأداة مبنية باستخدام ملف PDF يحتوي على أوصاف اللابتوبات!

🔢 التضمينات (Embeddings)

"Embeddings for this PDF are created and stored in a chroma vector database"
التضمينات لهذا الـ PDF تُنشأ وتُحفظ في قاعدة بيانات chroma vector!

🔍 الاسترجاع الذكي

"The retriever tool retrieves passages from the document, based on the input query"
أداة الاسترجاع تسترجع مقاطع من المستند، بناء على استعلام الدخل!

📄 المصدر

ملف PDF بمواصفات مفصلة

🔢 المعالجة

تحويل النص لـ embeddings

💾 التخزين

قاعدة بيانات Chroma Vector

🎯 الاسترجاع

بحث دلالي متقدم

قوة تقنية Vector Database:
• بحث دلالي (ليس مجرد كلمات مطابقة) 🧠
• فهم السياق والمعنى 💡
• استرجاع المعلومات الأكثر صلة 🎯
• سرعة عالية حتى مع كمية كبيرة من البيانات ⚡
سير العمل:
سؤال المستخدم → تحويل لـ embedding → بحث في قاعدة البيانات → استرجاع أقرب المقاطع → تقديم المعلومات الأكثر صلة
