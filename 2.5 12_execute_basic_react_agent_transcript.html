# Alternative using HumanMessage from langchain_core.messages import HumanMessage messages = [HumanMessage(content="What is the sum of 2 and 3?")]

"The question we ask is, 'What is the sum of 2 and 3?'"

💬 تتبع الرسائل:

👤 رسالة المستخدم (Human Message):
"The first message is the Human Message, or input, which is the original question to the agent"
الرسالة الأولى هي رسالة المستخدم، أو المدخل، واللي هي السؤال الأصلي للوكيل!
🧠 رسالة الذكاء الاصطناعي (AI Message):
"Next, we have the AI Message. This is the LLM executing the thought block"
بعدين، عندنا رسالة الذكاء الاصطناعي. ده الـ LLM بينفذ كتلة التفكير!
🔧 رسالة الأداة (Tool Message):
"The next message is the Tool Message. This is the output of the tool"
الرسالة التالية هي رسالة الأداة. ده مخرج الأداة!
🎯 رسالة الذكاء الاصطناعي النهائية:
"The last message is the AI Message. Here, the LLM looks at the results from the tool"
الرسالة الأخيرة هي رسالة الذكاء الاصطناعي. هنا، الـ LLM بيبص على نتائج الأداة!

🎯 التحدي الثاني

"Then, we execute a second input. Here, we provide a more complex prompt, 'What is 3 multiplied by 2 and 5 plus 1?'"

🎯 النتيجة الصحيحة

"We follow the same pattern to print the messages. We see that the LLM has correctly answered both the questions with the answer as 6"

تحليل الإجابة:
• 3 × 2 = 6 (باستخدام find_product)
• 5 + 1 = 6 (باستخدام find_sum)
• الإجابة النهائية: كلاهما = 6
