دليل شامل لتجهيز بيئة التطوير السحابية للكورس

☁️ إيه هو GitHub Codespaces؟

GitHub Codespaces ده بيئة تطوير سحابية جبارة! تخيل إنك تقدر تشتغل على أي مشروع من أي مكان في العالم من غير ما تحتاج تنصب حاجة على جهازك. ده زي إن عندك كمبيوتر قوي في السحابة جاهز للشغل في أي وقت.

ليه Codespaces مهم للكورس ده؟
لأن كل الأدوات والمكتبات اللي محتاجينها للـ AI agents هتكون جاهزة ومضبوطة، مش هتحتاج تدوخ نفسك في التنصيب والإعداد!

فكر فيه كده: بدل ما تقضي ساعات في تنصيب Python وLangChain وكل المكتبات دي، كل حاجة تكون جاهزة في دقائق معدودة!

🚀 سرعة التجهيز

كل الأدوات جاهزة في دقائق بدل ساعات من التنصيب

🔧 بيئة موحدة

كل الطلاب يشتغلوا على نفس البيئة بنفس الإصدارات

💻 مش محتاج جهاز قوي

يشتغل على أي جهاز حتى لو ضعيف، لأن الشغل في السحابة

🌐 اشتغل من أي مكان

محتاج بس إنترنت ومتصفح، وخلاص!

📋 خطوات الإعداد بالتفصيل

دلوقتي هنشوف مع بعض إزاي نجهز بيئة العمل خطوة بخطوة. المدرب شرح الموضوع بطريقة سهلة جداً، تعالوا نفهمها بالتفصيل:

1 الوصول للمستودع (Repository)
أول حاجة محتاجين نعملها إننا نروح لمستودع الكود بتاع الكورس اللي اسمه build-ai-agents-and-chatbots-with-langgraph. ده المكان اللي فيه كل الكود والأمثلة اللي هنشتغل عليها في الكورس.
2 فتح Codespaces
هنلاقي زرار اسمه "Code" في صفحة المستودع، نضغط عليه وبعدين نختار "Create Codespace on Main". ده هيبدأ يجهز لنا بيئة العمل في السحابة.
3 انتظار التجهيز
دلوقتي GitHub هيبدأ يجهز البيئة وينصب كل البرامج اللي محتاجينها. ده هياخد دقائق معدودة بس، وهنشوف شاشة التحميل وهي بتشتغل.
البيئة جاهزة!
لما التجهيز يخلص، هنلاقي IDE كامل جاهز للشغل في المتصفح، زي ما لو إحنا فاتحين VS Code على جهازنا!

📓 فتح الـ Notebook الأول

بعد ما البيئة تبقى جاهزة، المدرب قال إننا نروح نفتح الـ notebook الأول اللي اسمه:

📁 اسم الملف

code_02_XX Basic ReAct Agent in LangGraph.ipynb

ده أول notebook هنشتغل عليه في الكورس، وفيه أساسيات الـ ReAct Agent اللي هنتعلمها.

الـ notebook ده مهم جداً لأن فيه كل الأساسيات اللي هنبني عليها باقي المشاريع. وفيه أول cell (خلية) فيها كل المكتبات اللي محتاجينها.

ملاحظة مهمة:
لازم نشغل الـ cell الأولى في الـ notebook عشان ننصب كل المكتبات المطلوبة قبل ما نبدأ أي تمرين.

📦 تنصيب المكتبات المطلوبة

أهم خطوة في الإعداد هي إننا نشغل الـ cell الأولى في الـ notebook. الـ cell دي فيها كل الـ packages (المكتبات) اللي محتاجينها في كل تمارين الكورس.

إيه اللي هيحصل لما نشغل الـ cell الأولى؟
النظام هيبدأ يحمل وينصب مكتبات زي LangChain، LangGraph، وكل الأدوات اللي محتاجينها للـ AI agents.

المكتبات دي مش مجرد كود عادي، دي أدوات متقدمة جداً في عالم الذكاء الاصطناعي:

🦜 LangChain

مكتبة أساسية لبناء تطبيقات الذكاء الاصطناعي والتعامل مع نماذج اللغة

📊 LangGraph

أداة لبناء الـ workflows المعقدة وربط الـ agents مع بعض

🤖 OpenAI SDK

للتعامل مع نماذج GPT وباقي خدمات OpenAI

📚 مكتبات أخرى

زي pandas للبيانات، requests للاتصال، وغيرها من الأدوات المساعدة

⚡ تشغيل عملية التنصيب
المدرب قال "Go ahead and run these dependency installation" - يعني نضغط على زرار Run في الـ cell الأولى ونستنى لحد ما التنصيب يخلص.
